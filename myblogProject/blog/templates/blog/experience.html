Query-focused Multi Document Summarization
Description about the summarization system

This work is done to enable the users to gather information from multiple articles that are discussed about BP oil disaster that was occured on 20th of April 2010 in the Gulf of Mexico, United States. The dataset was created by Giang Binh Tran, Tuan A. Tran, Nam-Khanh Tran, Mohammad Alrifai and Nattiya Kanhabua (http://www.l3s.de/~gtran/publications/taia2013.pdf), they make it available on http://www.l3s.de/~gtran/timeline/

The user only need to generate the query regarding above disaster and inset it into the bellow bar. The system then gathers information from multiple docs and produces summary that satisfies the users needs. In addition, retrieval model, machine earning and AI methods were used in this work. ,
